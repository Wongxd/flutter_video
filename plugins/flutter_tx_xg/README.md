# tx_xg

A new flutter plugin project.
### use
in you project open `android/app/build.gradle`

```
manifestPlaceholders = [

        XG_ACCESS_ID: "YOU_XG_ACCESS_ID",
        XG_ACCESS_KEY: "YOU_XG_ACCESS_KEY",
]
```
